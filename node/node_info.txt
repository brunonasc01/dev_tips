:Para corrigir o /favicon.ico no request
Insira o codigo abaixo na criacao do servidor
    // control for favicon
    if (request.url == '/favicon.ico') {
        response.writeHead(200, {'Content-Type': 'image/x-icon'} );
        response.end();
        return;
    }
